## What is this repository?
This repository serves as the **specification and documentation hub** for the MindFirst Engine (M1E). It houses:

- **Architecture Specifications:** (Profiler, Interpreter, Stabiliser, Response Generator, API Layer)
- **Safety Constraints:** Post-identity and structure-only interaction rules.
- **Governance & Ethics:** Compliance frameworks and metadata standards.
- **Validation:** Evaluation frameworks and research rationale.
